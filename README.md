# YOUR-PERSONAL-CHATBOT

## How It's Made
**Tech Used:** Python, HTML, JavaScript, FastAPI 

## Introduction
YOUR-PERSONAL-CHATBOT is an AI-powered chatbot that helps users search their own documentation using natural language.

## Installation

```bash
git clone https://github.com/Melitou/YOUR-PERSONAL-CHATBOT
cd project
pip install -r requirements.txt
fastapi run master_pipeline.py
```
